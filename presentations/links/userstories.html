User Stories

-As a professor, I would like to create accounts for students

-As a professor, I would like to create accounts for TA's with permissions to create questions

-As a professor, I can create an assignment for each lesson/chapter with the start date and end date

-As a professor, I should be able to create questions such as multiple choice, text input with the correct solution and the grade weight to each assignment
-As a professor, I would like to be able to edit questions and answers after creation

-As a TA, I should be able to create questions such as multiple choice, text input with the correct solution and the grade weight to each assignment

-As a professor, I would like each assignment/quiz to be randomized with about 10 questions

-As a student, I would like to take a quiz (epic)

-As a student, I would like to be able to retake a quiz

-As a student, I would like to see my grades after I finish the quiz

-As a professor, I would like to see the students total grade for each quiz

-As a student, I would like to see feedback/ correct solution (after the deadline)

gamify-As a professor or TA, I should be able to set a difficulty level for each question

-As a student, I would like to be able to choose a difficulty level for the quiz

(edit/add stuff)-As a student, I should be able to see the leaderboard with a list of top students with the highest scores
